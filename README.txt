Required Dependencies:
  - python 3.x
  - numpy, panda, sci-kit learn, tabular, matplotlib

Commands:
  - python3 all_feat.py : Selecting all features
  - python3 PCA_feat.py : Feature selection using PCA
  - python3 RFE_feat.py : Feature selecting  using RFE
  - python3 Corr_feat.py : Feature selection using Correlation Values
